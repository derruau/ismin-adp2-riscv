# Informations

Ce dossier contient le code du TD 2.

Dans les dossiers `hdl_src/QX` vous trouverez le code tel qu'il était au moment de la question X du TD.

Le code qui se trouve directement dans `hdl_src` est le code de la dernière question. Donc il y a la même chose
dans `hdl_src/Q17` et dans `hdl_src`.

# Compilation du projet

Pour compiler le projet, vous pouvez utiliser, dans le dossier racine:
```bash
./build.sh
```

Vous pouvez toujours compiler en utilisant les commandes de base:
```bash
cd firmware
./build.sh
cd ../sim
./build.sh
```

Cela compilera la dernière version du projet.
